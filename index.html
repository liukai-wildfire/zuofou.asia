  • Social Network Connector

    Social Network Connector

    If you allow users to sign up through different social networks and perhaps your own registration system, there is a good chance some users will have multiple accounts. How annoying can it be for a user who signed up through Facebook earlier, to come back later and log in through Weibo because he thought he used that one?

    Features

    • Bind and undind SNS accounts.
    • merge social accounts so no need to maintain multiple accounts.
    • Category: Application, Widget
    • Services in project: coding
    • Date released: February 2013

  • JieWen Mobile App

    JieWen Mobile App

    JieWen is a better way to ask a question with photos, maps, friends, and more. It's also people helping each other.  

    Features

    • Ask questions with images, maps, and locations to deepen the context.
    • Answer map questions by dropping a pin for a specific location. Answer photo questions by drawing on them. Answer any question by attaching a link.
    • JieWen routes questions using your existing social network connections, location, and topics you are interested in.
    • Questions can be forwarded outside the app so your friends who don't have Jelly can still help.
    • Category: Mobile App
    • Services in project: Design, UX, Coding
    • Date released: Nov 2014
